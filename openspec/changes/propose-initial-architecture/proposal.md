# Proposal: Initial Application Architecture

This proposal outlines the initial software architecture for the shopping application. 

The goal is to establish a solid foundation based on the requirements specified in `doc/REQ.md`. The proposed architecture follows standard best practices for modern web applications, utilizing a layered approach with a Spring Boot backend.

This design aims to create a system that is scalable, maintainable, and testable.

---
# 提案：初始應用程式架構

本提案概述了購物應用程式的初始軟體架構。

目標是根據 `doc/REQ.md` 中指定的需求建立一個穩固的基礎。所提出的架構遵循現代 Web 應用程式的標準最佳實踐，採用分層方法並使用 Spring Boot 後端。

此設計旨在建立一個可擴展、可維護且可測試的系統。
