# 規格：行銷活動 (Marketing Campaigns)

## ADDED Requirements

### #### Requirement: `marketing-campaigns-1` 系統需實作「雞娃娃扭蛋機」活動
- **As a** 顧客
- **I want to** 在消費滿一定金額後，獲得一次抽獎機會
- **So that** 我能感受到購物的額外樂趣並有機會獲得優惠。

#### Scenario: 觸發扭蛋機
- **Given** 我是一個顧客，且「雞娃娃扭蛋機」的觸發門檻是 120 元
- **When** 我的購物車總金額達到 125 元並完成結帳
- **Then** 系統應自動跳出一個扭蛋機的互動介面，讓我進行抽獎。

#### Scenario: 未觸發扭蛋機
- **Given** 我是一個顧客，且「雞娃娃扭蛋機」的觸發門檻是 120 元
- **When** 我的購物車總金額為 110 元並完成結帳
- **Then** 系統應正常顯示訂單完成頁面，不會跳出扭蛋機介面。

### #### Requirement: `marketing-campaigns-2` 系統需實作「金蛋幸運日」活動
- **As a** 顧客
- **I want to** 在特定日子（例如每週一）獲得專屬優惠
- **So that** 增加我回訪的意願。

#### Scenario: 在幸運日獲得優惠
- **Given** 今天是星期一，且「金蛋幸運日」的優惠是「免費加一份布丁」
- **When** 我點購任一飲品
- **Then** 系統應在我的訂單中自動提供「免費加布丁」的選項或優惠券。

#### Scenario: 在非幸運日無法獲得優惠
- **Given** 今天是星期二
- **When** 我點購任一飲品
- **Then** 系統不應提供「免費加布丁」的優惠。
