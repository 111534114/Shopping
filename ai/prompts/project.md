1.專題名稱 / Project Title / 프로젝트명
雞娃娃飲料店線上點餐系統
2. 專案目的 / Purpose / 목적
提供雞娃娃飲料店顧客一個方便、快速的線上點餐平台，提升點餐效率並優化顧客體驗。
3. 主要使用者 / Target Users / 주요 사용자
- 雞娃娃飲料店的顧客：希望快速方便點餐、查詢菜單、追蹤訂單。
- 雞娃娃飲料店的員工：負責處理訂單、管理菜單、查看銷售數據。
4. 網站主要功能 / Key Features / 주요 기능
- 顧客端：
    - 瀏覽菜單與商品詳情
    - 加入購物車、調整數量與選擇配料
    - 線上支付功能
    - 訂單查詢與追蹤
    - 會員註冊與登入
- 店家端：
    - 訂單管理（接單、備貨、完成）
    - 菜單管理（新增、修改、刪除商品與配料）
    - 銷售數據報表
5. 成功指標 / Success Metrics / 성공 지표 
- 顧客滿意度提升：透過問卷調查或評論，達到80%以上滿意度。
- 線上訂單比例：上線後三個月內，線上訂單佔總訂單量20%以上。
- 訂單處理效率：平均訂單處理時間縮短15%。
- 系統穩定性：系統正常運行時間達到99.5%以上。
6. 非範圍內容 / Out of Scope / 범위 밖
- 實體店面POS系統整合
- 物流配送功能
- 多國語言支援
- 複雜的促銷活動模組（例如：買一送一、第二件半價等）
- 顧客評論與評分系統
