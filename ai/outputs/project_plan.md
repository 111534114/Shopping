1.專題名稱 / Project Title / 프로젝트명
雞娃娃飲料店線上點餐系統

2. 專案目的 / Purpose / 목적
- 提供顧客一個方便快捷的線上點餐管道，減少現場等候時間。
- 提升店家處理訂單的效率，並降低人工接單的出錯率。
- 透過線上平台，店家可以更輕鬆地推出促銷活動與管理會員。
- 收集線上訂單數據，以利分析顧客偏好與熱銷商品。

3. 主要使用者 / Target Users / 주요 사용자
- **顧客 (Customers):** 習慣使用手機點餐的學生、上班族或附近居民，追求方便與效率。
- **店家管理員 (Admin):** 負責管理菜單、查看訂單、分析銷售數據的店長或員工。
- **店員 (Staff):** 在店內接收並處理線上訂單的員工。

4. 網站主要功能 / Key Features / 주요 기능
- **前台 (顧客端):**
    - **菜單瀏覽:** 分類顯示所有飲品，包含品項名稱、價格、圖片與描述。
    - **客製化選項:** 提供甜度、冰塊、加料等客製化選項。
    - **購物車功能:** 可新增、修改、刪除訂單中的商品。
    - **線上結帳:** 支援常見的支付方式 (如信用卡、LINE Pay、街口支付)。
    - **訂單查詢:** 顧客可查看歷史訂單與目前訂單的處理狀態。
    - **會員系統:** 簡易的註冊/登入功能。

- **後台 (店家端):**
    - **訂單管理:** 即時接收新訂單通知，並更新訂單狀態 (如：準備中、已完成、待取餐)。
    - **菜單管理:** 可新增、編輯、刪除或暫時下架菜單上的品項與分類。
    - **銷售報表:** 提供簡單的日、週、月銷售數據統計與圖表。

5. 成功指標 / Success Metrics / 성공 지표
- **上線後三個月內，線上訂單量佔總訂單量的 20% 以上。**
- **顧客滿意度調查達到 4.5 顆星 (滿分 5 顆星)。**
- **每日透過線上系統完成的訂單平均超過 50 筆。**
- **註冊會員數達到 1000 人。**

6. 非範圍內容 / Out of Scope / 범위 밖
- **外送功能:** 初期版本專注於線上點餐、到店自取。不包含與外送平台的整合。
- **複雜的會員/點數系統:** 暫不開發會員等級、點數累積與兌換等功能。
- **庫存管理系統:** 暫不與庫存連動，品項的供應狀態由店家手動管理。
- **多國語言支援:** 初期僅提供正體中文介面。
